üîß COMMON COMMANDS
================================================================================

Start Services
  docker-compose up -d

Stop Services
  docker-compose down

View Logs
  docker-compose logs -f

Check Status
  docker-compose ps

Create Admin User
  docker exec backend python -c "
  from db.session import SessionLocal
  from db.models import User
  from api.auth import get_password_hash
  from uuid import uuid4
  
  db = SessionLocal()
  user = User(
    id=uuid4(),
    email='admin@example.com',
    hashed_password=get_password_hash('password123'),
    full_name='Admin User',
    role='admin'
  )
  db.add(user)
  db.commit()
  print('Admin user created')
  "

Run Tests
  docker exec backend pytest tests/

Backup Database
  docker exec postgres pg_dump -U user email_marketing > backup.sql

üÜò TROUBLESHOOTING
================================================================================

Services Won't Start
  1. Check Docker is running
  2. Check ports not in use (8000, 3000, 5432, 6379)
  3. Check disk space (10GB+)
  4. Try: docker-compose down && docker-compose up

Can't Login
  1. Check admin user created
  2. Verify email/password
  3. Check database connection
  4. Recreate admin user (see COMMANDS above)

Port Already in Use
  Windows: netstat -ano | findstr :8000
  Mac/Linux: lsof -i :8000
  Then kill the process

Database Connection Error
  docker-compose down
  docker volume rm email-marketing_postgres_data
  docker-compose up

Services Not Responding
  docker-compose logs -f
  docker-compose restart
