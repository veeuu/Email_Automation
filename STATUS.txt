================================================================================
üîß TROUBLESHOOTING
================================================================================

If backend won't start:
  1. Check port 8000 is not in use
  2. Delete email_marketing.db and restart
  3. Check backend/venv/Scripts/python works

If frontend won't start:
  1. Check port 3000 is not in use
  2. Delete frontend/node_modules and run npm install
  3. Check node --version shows v18+

If can't login:
  1. Check backend is running (http://localhost:8000/health)
  2. Check frontend is running (http://localhost:3000)
  3. Try refreshing the page
